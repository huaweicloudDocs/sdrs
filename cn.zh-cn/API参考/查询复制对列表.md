# 查询复制对列表<a name="sdrs_05_0603"></a>

## 功能介绍<a name="zh-cn_topic_0079693002_section34649765"></a>

查询指定保护组下的所有复制对列表，如果不给定指定保护组则查询当前租户下的所有复制对列表。

## 约束与限制<a name="section8242191544111"></a>

无
