# 添加保护实例标签<a name="ZH-CN_TOPIC_0151493877"></a>

## 功能介绍<a name="section156711130476"></a>

一个保护实例上最多有10个标签。

此接口为幂等接口：

创建时，如果创建的标签已经存在（key相同），则覆盖。

## URI<a name="section196717394711"></a>

-   URI格式

    POST /v1/\{project\_id\}/protected-instances/\{protected\_instance\_id\}/tags
