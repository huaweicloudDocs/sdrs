-   响应样例

    ```
    {
        "versions": [
            {
                "id": "v1",
                "links": [
                    {
                        "href": "https://sdrs.localdomain.com/v1",
                        "rel": "self"
                    }
                ],
                "status": "CURRENT",
                "updated": "2018-05-30T15:00:00Z",
                "version": "",
                "min_version": ""
            }
        ]
    }            
    ```

    或

    ```
    {  
          "error": {  
              "message": "XXXX",   
              "code": "XXX"  
          }  
      }
    ```

    其中error是泛指的错误，有badrequest、itemNotFound等，如报错为：

    ```
    {  
          "badrequest": {  
              "message": "XXXX",   
              "code": "XXX"  
          }  
      }
    ```
